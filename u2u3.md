# Unit 2 - 机会成本
生产更多个单位的B放弃掉多少单位个A(单位统一)  
广义上代表任何放弃的东西

- 机会成本的变化反应消耗资源类型的相关性  
如果机会成本一样 代表放弃多少个资源给B A就能利用到多少资源
有点像`物质守恒` 如果变了证明资源类型不一样 比如塑料/金属

- 对于货物A,B 机会成本P   
假设 $P_a = x $  
那么这时 $P_b = \frac{1}{P_a}$  
一个国家不可能同时获得两个比较优势, 对于国家甲、乙:  
如果 $P_{a甲} < P_{a乙}$  
根据倒数关系 对于货物b的成本  
$P_{b甲} = \frac{1}{P_a甲} > \frac{1}{P_a乙}=P_{b乙}$  
但可能出现两个国家比较优势相同的情况

**“凹”（concave）和“凸”（convex）**

# Unit 2 - 贸易

- 绝对优势: 更低的成本 比如A生产木头需要1h, B需要0.5h  
相同的成本不等于绝对优势

- 比较优势: 不同的机会成本  
国家应该生产比别的国家机会成本更低的商品

- 只要贸易条件(交换比率、报价)介于两国各自的机会成本之间，双方都能获益。不存在一方获益必须以另一方受损为前提。
如果进口国报价<出口国机会成本: 换另一个物品不如自己生产
如果出口价格>进口国机会成本：花钱买不如自己生产

- 即使有绝对优势，也该有贸易。只看比较优势  
贸易是个正和游戏 没有零和结果

---

稀缺性​ -> 选择生产什么​ -> 机会成本​ -> 用生产可能性曲线量化 -> 不同机会成本的比较优势​ -> 贸易​ -> 获益
